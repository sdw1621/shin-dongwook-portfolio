# 신동욱 | AI 교육 전문가 포트폴리오

생성형 AI · 빅데이터 · 프로그래밍 교육 전문가 **신동욱** 강사의 포트폴리오 사이트입니다.

## 구성
- `index.html` — 메인 (히어로 + 핵심 지표)
- `about.html` — 소개
- `skills.html` — 전문 분야
- `experience.html` — 주요 이력
- `clients.html` — 협업 기관
- `publications.html` — 저서 & 연구
- `contact.html` — 문의하기

## 기술
- 정적 HTML (멀티 페이지)
- Tailwind CSS (CDN) · Pretendard 폰트
- 이미지는 base64로 내장되어 외부 의존성 없음
